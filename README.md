# That's My Dogg Southern BBQ website

A responsive, one-page restaurant website built with plain HTML, CSS, and JavaScript.

## Preview
Open `index.html` in a browser.

## Publish
Upload the contents of this folder to any static host, including Netlify, Cloudflare Pages, GitHub Pages, cPanel, or a normal web server.
